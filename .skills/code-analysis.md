# Code Analysis Skill

Analyze codebases to identify patterns, quality issues, and improvement opportunities.

## Workflow

1. Scan codebase structure using codebase_search
2. Identify code patterns and anti-patterns
3. Check for linting errors using read_lints
4. Analyze code quality metrics
5. Generate analysis report
6. Suggest improvements

## Examples

- Analyze entire project for code quality
- Find all instances of a specific pattern
- Identify unused code
- Check for security vulnerabilities
- Measure code complexity
