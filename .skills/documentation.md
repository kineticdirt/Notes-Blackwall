# Documentation Skill

Generate comprehensive documentation for projects, APIs, and codebases.

## Workflow

1. Analyze codebase structure
2. Extract docstrings and comments
3. Identify undocumented areas
4. Generate API documentation
5. Create usage examples
6. Update README and guides

## Examples

- Generate API documentation from code
- Create user guides
- Document workflows and processes
- Update README files
- Generate architecture diagrams (text-based)
